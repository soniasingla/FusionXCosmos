## 🚨 Troubleshooting

### Common Issues

**"Insufficient ETH sent" Error:**
```bash
# Check safety deposit requirement
node -e "
const { ethers } = require('hardhat');
// Check contract requirements and adjust accordingly
"
```

**Neutron RPC Issues:**
- Try alternative RPC: `https://rpc.neutron.nodejumper.io`
- Check network status: [Neutron Status](https://neutron.celat.one/pion-1)

**Transaction Pending:**
- Sepolia can be slow - wait 2-3 minutes
- Check mempool: [Sepolia Mempool](https://sepolia.etherscan.io/txsPending)

### Network Alternatives
If testnets are slow:
- Ethereum: Switch to Goerli (if configured)
- Cosmos: Switch to other test networks (if configured)
